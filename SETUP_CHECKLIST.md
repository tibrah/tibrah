# Volunteer Setup Checklist

Welcome! Follow these steps to get started contributing to the AIGS Website Project.

---

## 1. Access
- [ ] GitHub account created
- [ ] Added to the repository
- [ ] Access verified

---

## 2. Local Development Setup
- [ ] Install Node.js (v18+)
- [ ] Clone the repository
- [ ] Install dependencies:
      npm install
- [ ] Start local server:
      npm run dev
- [ ] Confirm site runs locally

---

## 3. CMS Integration Check
- [ ] Confirm Sveltia admin loads at /admin
- [ ] Review CMS schemas under /cms or /content
- [ ] Check editor preview

---

## 4. Tools & Workflow
- [ ] Review CONTRIBUTING.md
- [ ] Review labeling system
- [ ] Join GitHub board for tasks
- [ ] Assign yourself a “good-first-issue” or open task

---

## 5. Contribution Workflow
- [ ] Create feature branch: feature/your-task
- [ ] Make changes
- [ ] Open Pull Request
- [ ] Request review
- [ ] Merge after approval

---

## 6. Communication
- [ ] Join the team Slack/Discord
- [ ] Attend check-ins (when available)

Thank you for contributing!
